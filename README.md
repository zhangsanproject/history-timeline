# 华夏风云纪事
更新
一个沉浸式中国历史时间轴网页，以滚动动画的形式呈现从安史之乱到崖山海战五百年间的重大历史事件（公元 755 年 — 1279 年）。

## 项目简介

本项目是一个纯前端静态网页，无需任何构建工具或后端服务。打开 `index.html` 即可在浏览器中运行。

时间轴涵盖了以下历史阶段的关键事件：

- **盛唐衰落**：安史之乱、黄巢起义、白马驿之祸
- **五代更迭**：后梁、后唐、后晋、后汉、后周的兴衰
- **十国兴废**：吴越、南唐等割据政权
- **北宋一统**：陈桥兵变、杯酒释兵权、统一战争
- **北宋风云**：澶渊之盟、庆历新政、王安石变法、党争
- **靖康之变**：金灭辽、靖康之耻、北宋灭亡
- **南宋抗争**：岳飞抗金、郾城大捷、采石矶之战
- **宋元易代**：金国灭亡、元军南侵、崖山海战

## 技术栈

- **HTML5** — 页面结构
- **CSS3** — 时间轴布局与滚动动画（`transition` + `transform`）
- **JavaScript** — 基于 `IntersectionObserver` 思路的滚动触发动画（`scroll` 事件监听）

## 文件结构

```
├── index.html    # 页面主体，包含全部时间轴节点数据
├── style.css     # 时间轴样式与动画效果
├── script.js     # 滚动监听与节点显示逻辑
└── README.md
```

## 使用方式

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。

```bash
# macOS
open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```

## 特色

- 左右交错的卡片布局，时间轴主轴居中
- 滚动时节点逐个淡入，沉浸式浏览体验
- 纯原生实现，零依赖，加载极快
